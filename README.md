# der-pw.github.io

This is the repo for [der-pw.github.io](https://der-pw.github.io).

## Projekt lokal starten

### Voraussetzungen

- Node.js
- npm

### Abhaengigkeiten installieren

```bash
npm install
```

### Entwicklungsserver starten

```bash
npm run docs:dev
```

Danach ist das Projekt lokal ueber die von VitePress ausgegebene URL erreichbar, in der Regel `http://localhost:5173`.

### Produktions-Build erstellen

```bash
npm run docs:build
```

### Produktions-Build lokal testen

```bash
npm run docs:preview
```
